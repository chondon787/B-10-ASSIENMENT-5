What is JavaScript, and how is it different from Java?

JavaScript is for web interactivity, while Java is used for large applications.
They have different use cases and structures despite similar names.

Explain the difference between let, const, and var.

var: Function-scoped, re-declarable.
let: Block-scoped, re-assignable, not re-declarable.
const: Block-scoped, immutable reference, not re-assignable.

What is the difference between == and === in JavaScript?

==: Allows type conversion.
===: No type conversion, strict comparison.
